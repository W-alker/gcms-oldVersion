# 国家大创 - 垃圾分类管理系统（GCMS）
> 大创项目。一个垃圾分类管理系统（Garbage classification management system）。
## 关于作者
**团队人员**：安徽三联学院，程旭、梅林、张俭。
<br/>
**人员负责**：所有代码实现部分由本人（程旭）一人完成。项目计划书等由其他二人负责。
## 技术栈
**前端**：`HTML`+`SCSS`+`JQ`+`BootStrap`
<br/>
**后端**：`Node.js`+`Express`
<br/>
**数据库**：`Mysql`
<br/>
**自动化工具**：`Gulp`
<br/>
